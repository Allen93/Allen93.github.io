写与2018年底

总结

2018工作依然很忙，但是非常开心的是工作之余，坚持做了几件有意思的事。这些事给我带来的满足远超预期。由于一些原因，对工作和个人成长目标等也有了一些思考。

满足的事情

  • 坚持健身半年
  • 蛙泳单程50米吃力 -》 200米不吃力
  • 初学自由泳 25米….
  • 专业外阅读（人类简史，人类动物园，活出生命的意义，激荡三十年，原则 等）重建了阅读习惯
  • 播客，坚持听了两个很棒的中文播客（黑水公园，得意忘形）
  • 删了知乎

领悟

  • 工作，一定要认清目标，想清楚目的。将核心诉求作为最高优先级执行。
  • 有了扩大影响力的意识，方式： 写文章，技术分享。

认识到可以提高

  • 英文阅读能力急切需要提高
  • 写文章能力要加强
  • 试着画一些架构图，需要精进，画的好看一点…
  • 更好的做设计
  • 整理学习知识

展望

展望下新一年的计划，按照我的风格，从小目标开始

技术

  • 做好一件小事
    • 加强架构设计能力， 做一次让自己满意的设计。从执行到结果。
    • 做一次让自己满意的分享，准备充分到让自己感觉到诉说欲爆棚。
  • 完成一些大事
    • 写好代码，拥有让自己满意的Coding能力。
    • 增强影响力
      • 至少在2个方向上产出几篇可以作为系列的文章。
      • 各类的总文章数不低于12篇。每月至少一篇吧
    • 英文阅读
      • P1 Kubernetes 官方文档精读
      • P1 GoLang 文档精读
      • P1 Google三架马车 其实挺容易的…
      • Tensorflow 六篇主要论文 （目前完成半篇。。）
    • 技术领域
      • P1 Kubernetes 精通，Tensorflow 分布式领域做到熟悉。
      • P1 Kubeflow 社区里发声
      • P2 Kubernetes 调度器精通
      • P2 ETCD
      • P2 调度器： Mesos Yarn
      • P2 计算框架： flink Spark Hadoop
      • P2 TCP/IP 协议
      • P3 SSL协议
      • P3 学习Haskell 或者Rust 选一个吧

个人

  • 健身
    • P1 坚持健身一年
    • P1 能一次完成5个以上引体向上
    • P1 能一次完成40个以上俯卧撑
    • P1 完成跑步10公里
    • P1 蛙泳达到单程500米（现200米）
    • P1 学会自由泳，单程200米
    • P2 体脂率 20%
    • P2 完成跑步20公里
  • 出行
    • 恢复骑车爱好，参加北京的车友活动。全年1000公里吧
    • 学车，完成一次自驾出行
    • 买帐篷，参加野营活动
    • 滑雪2次
    • 一次欧洲出行
    • 一次亚洲出行
    • 成都旅行
  • 理财
    • 花销规划
    • 准备20k用于较高风险投资，100k用于低风险理财
    • 尝试做投资分析
    • 坚持记账一个月
  • 身体
    • 尝试坚持一个星期的健康饮食
    • 尝试一个月内坚持至少15天早睡早起（11:30前睡）
  • 心理
    • 坚持一周冥想
    • 尝试一次心理咨询周期
